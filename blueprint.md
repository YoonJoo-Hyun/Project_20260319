# Lotto Number Generator

## Overview

A simple web application to generate and display random lottery numbers. The application will be built with HTML, CSS, and JavaScript, following modern web standards.

## Design and Features

*   **UI:** A clean, modern interface with a button to generate numbers and a visually distinct area to display the results.
*   **Header:** A prominent "Lotto Number Generator" title.
*   **Number Display:** Generated numbers will be shown in individual, styled circles.
*   **Button:** A "Generate Numbers" button with interactive styles.
*   **Animation:** A subtle animation for the numbers when they are generated.
*   **Color Palette:** A vibrant and energetic color scheme.
*   **Texture:** A subtle noise texture on the background for a premium feel.
*   **Shadows:** Soft, multi-layered drop shadows to create a sense of depth.
